Initiation à la Programmation Web

Formation complète en HTML5 et CSS3 pour débutants. Apprenez à créer des sites web modernes et responsives.

Voir le Programme

Objectifs de la Formation

HTML5

Comprendre la structure d’une page web et maîtriser les balises essentielles.

CSS3

Styliser des pages web avec couleurs, polices, marges et animations.

Responsive

Créer des interfaces adaptables aux téléphones, tablettes et ordinateurs.

Programme de la Formation

Jour 1 : HTML5

  • Structure d’une page HTML
  • Balises sémantiques
  • Listes, liens et images
  • Formulaires et tableaux

Jour 2 : CSS3

  • Sélecteurs CSS
  • Box Model
  • Positionnement
  • Flexbox

Jour 3 : Projet Final

  • Création d’une Landing Page
  • Responsive Design
  • Animations CSS
  • Publication GitHub Pages
